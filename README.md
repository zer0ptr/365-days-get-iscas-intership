# 365天获得中国科学院软件研究所实习
> 受Vancir师傅的[365天获取玄武实验室的工作](https://github.com/Vancir/365-days-get-xuanwulab-job)启发，此先又联系了所里某组的几位在读学生以及对他们的研究方向很感兴趣，投递简历后也得知他们愿意让一个高中在读的学生有机会去学习，于是乎我也开始尝试去做这样的一件事：记录从我刚开始学习二进制（Pwn, IoT）以及其他我感兴趣的例如“代码生成安全”这一领域自己的一些笔记，这些笔记都很零散也没有多少很有含量的内容，但对于这个年龄段的我而言是少有能让我体会到充实感的方式了，也希望各位前辈师傅们不啬指教，也期待各位有着同样兴趣的朋友跟我交流~

## 这是什么？
这是一份我给自己365天内获得中科院软件所某课题组实习工作所定下的学习清单/笔记，用于记录我这一段时间内的收获，也用于回顾来发现自己的不足，纵使最终我没有成功去实习，但这个过程对我来说也是快乐的，又因为时常有新的点去关注, 所以很多时候学习的内容并不连贯甚至于碎片化, 这也是无可避免的，加上本身我就是个蠢猪(bushi)也不指望有人能以这份笔记的路线去学习。
> 本身作为高中生我用于学习安全的时间也很碎片化，于是清单中的学习进度花费时间或许也会显得很长效率低以及时间不连贯QAQ.  
甚至看着是连贯的数字可能实际中间隔了5天（周一到周五在上学www）
